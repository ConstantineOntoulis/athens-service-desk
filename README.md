# Athens Service Desk

Athens Service Desk is a full-stack portfolio project built with ASP.NET Core Web API, SQL Server, Entity Framework Core, Docker, testing, and a future TypeScript frontend.

The application models a service-request workflow where users can submit requests, staff can process assigned work, managers can monitor workload, and administrators can manage system data.

## Current Status

Project planning and repository setup are in progress.

## Planned Stack

- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server
- xUnit
- Docker
- GitHub Actions
- Angular or React with TypeScript

## Planned Features

- User authentication
- Role-based authorization
- Service request creation and tracking
- Staff assignment workflow
- Request status history
- Appointment scheduling
- Dashboard summaries
- API validation and error handling
- Unit and integration tests
- Dockerized local setup
